Git commands i have learned so far:
1. git init command is used to initialize or install git to your folder to allow edits or changes to be tracked
2. git add command is used to track or stage any changes made
3. git commit -m saves or takes a snapshot of your work or 
4. git status lets the user know any unstaged changes on the branch brfore committing
5. git log shows your commit history of the project
6. git show displays contents of your commits in the terminal
7. git checkout displays content of a commit in the vscode terminal
8. git switch and commit hash used to move or switch to a specific commit
